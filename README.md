# Sum Game

> Build with [React](https://github.com/facebook/react) and [Redux](https://github.com/reduxjs/redux).

## Getting started

**Clone the repo**

```sh
git clone https://github.com/fardhanardhi/sum-game.git
```

**Jump to newly created folder**

```sh
cd sum-game
```

**Install packages**

```sh
npm install
```

```sh
npm start
```

Play it!
